## 💼 주요 프로젝트

### 🅿️ Parking Ticket System  
> Redis Sorted Set을 이용한 **선착순 주차권 신청 시스템**  
- 수백 명의 동시 요청을 안정적으로 처리  
- peek-pop 구조를 통한 데이터 유실 방지  
- 기술 스택: Spring Boot, Redis, MySQL, JPA

### 🍽️ EATceed – 잔반을 줄이는 스마트 식권 시스템  
> 에코노베이션 팀 프로젝트  
- 사용자 예약, 통계 기반 API 개발  
- Spring Security 기반 JWT 인증 설계
