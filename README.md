## 📂 주요 프로젝트

### 🅿️ Parking Ticket System
> 실시간 트래픽이 몰리는 주차권 선착순 신청 시스템

- Redis Sorted Set을 활용해 요청 순서 보장 및 동시성 제어
- peek-pop 구조의 배치 처리로 데이터 유실 방지
- 사용자 수백 명의 동시 요청 환경에서 안정적인 분배 처리 구현
- `Spring Boot`, `Redis`, `JPA`, `MySQL`, `Docker`
